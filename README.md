# Personalized Learning Path Recommender

## Overview
This project recommends a personalized learning path for students based on their skill level and interests.

## Features
- Skill-based course recommendations
- Beginner, Intermediate, Advanced levels
- Quiz-based assessment
- Progress tracking

## Technologies Used
- Python
- Streamlit
- Pandas
- Machine Learning Concepts

## How to Run the Project

1. Clone the repository

git clone https://github.com/yourusername/learning-path-recommender

2. Install dependencies

pip install -r requirements.txt

3. Run the application

streamlit run app.py

## Dataset
The dataset contains courses categorized by skill and level.
